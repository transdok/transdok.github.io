Meine KlientInnen

Meine KlientInnen

Im Programmbereich Pflege Menüpunkt Meine Klienten befindet sich eine Übersicht über alle geplanten Hausbesuche der nächsten Tage. Durch Klicken auf den entsprechenden Tag können Sie markieren, welche Aufgaben Sie übernehmen werden.

Das System zeigt dabei KlientInnen, die Sie in letzter Zeit betreut haben, zuerst an.

In der Auswahlleiste können Sie nun in der Ansicht Meine KlientInnen diese Klienten schnell finden.

Andere einteilen

Wenn erforderlich, können KlientInnen auch anderen MitarbeiterInnen zugeteilen werden. Gehen Sie dazu in den Programmbereich Arbeitsteilung Menüpunkt Zuordnung.
