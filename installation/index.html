Installation

Installation hier starten (Internet Explorer) oder Setup verwenden.

Hardware-Voraussetzungen

  • mindestens Intel Core i3
  • mindestens 4 GB Arbeitsspeicher
  • Bildschirmauflösung mindestens 1440x900 oder 1280x1024, je größer der Bildschirm desto besser
  • Festnetz-Internetanbindung ohne Datenmengeneinschränkungen, je schneller das Internet desto besser
  • empfohlen SSD oder Hybrid-Festplatte
  • empfohlenes Beispiel für Neukauf: Markenkomplettsystem, i5 (aktuelles Modell), 8 GB RAM, 128 GB SSD Festplatte

Software-Voraussetzungen

  • Betriebssystem Windows 7, Windows 8.1 oder Windows 10
  • Alle vorhandenen Servicepacks müssen installiert sein

Funktionsumfang

  • Der Funktionsumfang wird durch laufende Updates erweitert
  • Der aktuelle Stand kann der Leistungsbeschreibung entnommen werden
